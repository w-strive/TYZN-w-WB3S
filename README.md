# -w-WB3S--
&amp;gt; **V1.0功能说明(当前)：**  
* 1、1.3寸oled显示时间日期、温度、湿度、电量等数据 
* * 2、按键进行相关功能设置 
* * 3、使用DS3231作为时钟IC，方便离线时提供较准确的时间 
* * 4、使用SHT30作为温湿度传感器，动态性较好，精度较高 
* * 5、使用VEML6030对周围环境光进行采样，实现oled亮度自动调节，夜间熄屏运行 
* * 6、通过涂鸦智能的WB3S无线模块实现APP控制及查看当前环境数据
